## ✨ Overview

**Merge.AI** is a modern, lightweight web solution that combines:

- 🌐 Landing Page (waitlist + product showcase)  
- 💼 Careers Site (job listings + filtering)  
- 📝 Job Application System  
- 🤖 AI-Powered Employee Onboarding App (React)  

Built for **speed, simplicity, and scalability**, with zero backend required for core functionality.

## ⚡ Features

### 🌐 Landing Page
- Waitlist form (Google Forms integration)
- Animated UI (glassmorphism + gradients)
- Modal-based UX (FAQ, video, footer content)
- Live waitlist counter + interactions

### 💼 Careers Page
- Dynamic job filtering (department, location)
- Job detail modal
- Application submission flow

### 📝 Application Form
- Multi-section structured input
- Validation + progress tracking
- Resume upload support

### 🤖 Onboarding App (React)
- Dashboard with onboarding progress
- Task checklist (Day 1 → Month 1)
- Resource hub + team directory
- AI onboarding assistant
